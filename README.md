# Bikeramp-front

## About

### Description

Vue.js front-end application for [Bikeramp](https://github.com/jedrekdomanski/bikeramp).

### Technology

| Technology   | Version |
| :----------- | :-----: |
| Vue.js       |  2.4.4  |
| VueX         |  3.0.0  |
| VueRouter    |  3.0.1  |
| Axios        |  0.18.0 |

---

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
